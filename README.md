# Smart Financial Health & Micro-Savings Platform

## Overview

A fintech web application that helps users track expenses, improve financial habits, and automate small savings using intelligent insights and alerts.

---

## Features

* User registration and profile management
* Dashboard with balance, transactions, and financial health score
* Expense tracking with automatic categorization
* Spending analytics (monthly and category-wise)
* Smart alerts (overspending and bill reminders)
* Micro-savings (round-up savings and goal tracking)

---

## Financial Health Score

Calculated based on:

* Savings ratio
* Spending behavior
* Bill payment consistency

---

## Tech Stack

* Zoho Creator (Frontend and Backend)
* Deluge scripting
* Built-in database

---

## Automation

* Automatic financial score updates
* Alerts triggered on spending limits
* Dynamic insights generation

---

## Purpose

To build a user-centric fintech solution focused on financial awareness and smart saving rather than just transaction tracking.
